# Task: count vowels

Write an Aether function `countVowels(s: String) returns Int` that returns the
number of vowels (lowercase a, e, i, o, u) in the string. Lowercase the input
first so capital vowels also count.

Then in `main`, print the count for each of the strings, one per line:

    "hello"
    "AEIOU"
    "rhythm"
    "The Aether Programming Language"

Expected output:

    2
    5
    0
    11
